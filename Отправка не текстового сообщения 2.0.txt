функциональные требования для отправки не текстовых сообщений 

---------------------- Аудиозапись ---------------------------

- Возможность записи аудиосообщения во время диалога.Для этого
нужно удерживать кнопку "Микрофон",что бы запись началась и
появился вспомогательный пользовательский интерфейс для удобства
аудиозаписи.Кнопка "Аудиозапись" находится справа от поля ввода сообщения.

	- При удерживании кнопки "Микрофон" производится запись аудиосообщения.
	При убратии пальца,запись будет прекращена, и будет автоматически отправлена 
	в ветку диалога.

	- Удаление записываемой аудиозаписи,производится посредством уже 
	нажатого пальца на кнопку"Микрофон" и его проведением влево,после чего 
	запись будет удалена.

	- Запись аудиосообщения без удерживания кнопки "Микрофон",производится
	посредством уже нажатого пальца на кнопку"Микрофон" и его проведением влево,
	после чего палец можно убрать.

------------------------- Камера -----------------------------

- При нажатии кнопки "Камера",пользователю откроется камера с упрощенным
пользовательским интерфейсом камеры,для возможности сделать фото.В число 
пользовательского интерфейса входят кнопка "сфотографировать",кнопка
"сменить камеру" и строка галереи,где отображаются последние сделаные фото.
Кнопка "Камера" находится справа от поля ввода сообщения.

	- При нажатии на кнопку "Сфотограировать" будет сделана фотография,после 
	чего ее можно будет предварительно посмотреть или удалить,а так же оставить
	подпись перед отправкой в ветку диалога 

	- При удерживании кнопки "Сфотографировать" будет включена функция
	записи видео.При убратии пальца видео будет записано.После 
	записи видео,его можно будет предварительно просмотреть или удалить,а так же
	оставить подпись перед отправкой в ветку диалога 

	- В пользовательском интерфейсе "Камера" можно сменить камеру с основной
	на фронтальную,и наоборот нажатием на кнопку "Сменить камеру".

	- В пользовательском интерфейсе "Камера" можно выбрать фото из галереи,
	нажатием на фото из строки галереи.Для открытия всего списка фотографий,
	нужно провести палецем вверх по экрану.

------------------------ Прикрепить ------------------------------

- При нажатии кнопки "Прикрепить" пользователю откроется всплывающее
окно с кнопками для выбора данных которые он может отправить в ветку
диалога.Таких кнопок 6,а именно:

	- "Документ".При нажатии на кнопку "Документ" откроется
	Пользовательский интерфейс (файловый менеджер)со всеми имеющимися
	документами для отправки.

	- "Камера".При нажатии кнопки...смотреть "Камера"

	- "Галерея".При нажатии кнопки "Галерея" пользователю
	открывается галерея со всеми фотографиями пользователя для 
	отправки.

	- "Аудиофайлы".При нажатии кнопки "Аудиофайлы" пользователю
	Открывается стандартный аудиоплеер со списком аудиозаписей для отправки

	- "Мое местоположение".При нажатии на кнопку "Мое местоположение"
	пользователю,при наличии включенной геолокации и при наличии интернета,
	отрывается карта с его местоположением для отправки

	- "Контакты".При нажатии на кнопку "Контакты" пользователю откроются
	все его ранее записанные контакты для отправки
