функциональные требования для отправки не текстовых сообщений 

--------------------------- Аудиозапись ---------------------------

- Должна быть возможность записи аудиосообщения во время диалога,для сокращения времени
потраченого на набор текста,или просто для удоства общения.
Для этого надо удерживать кнопку "Микрофон",что бы запись началась и
появился вспомогательный пользовательский интерфейс для удобства
аудиозаписи.Кнопка "микрофон" находится справа от поля ввода сообщения.

	- При удерживании кнопки "Микрофон" производится запись аудиосообщения.
	При убратии пальца,запись будет прекращена, и будет автоматически отправлена 
	в ветку диалога.

	- Удаление записываемой аудиозаписи,производится посредством уже 
	нажатого пальца на кнопку"Микрофон" и его проведением влево,после чего 
	запись будет удалена.

	- Запись аудиосообщения без удерживания кнопки "Микрофон",производится
	посредством уже нажатого пальца на кнопку "Микрофон" и его проведением вверх,
	после чего палец с кнопки "микрофон" можно убрать.

----------------------------- Камера -----------------------------

- Должна быть возможность сделать фото/видео для собеседника не выходя из диалога.
При нажатии кнопки "Камера",пользователю должна открыться камера с упрощенным
пользовательским интерфейсом камеры,для возможности сделать фото.В число 
пользовательского интерфейса входят кнопка "сфотографировать",кнопка
"сменить камеру" и строка галереи,где отображаются последние сделаные фото.
Кнопка "Камера" находится справа от поля ввода сообщения.

	- При нажатии на кнопку "Сфотографировать" должна быть сделана фотография,после 
	чего ее можно будет предварительно посмотреть или удалить,а так же оставить
	подпись перед отправкой в ветку диалога 

	- При удерживании кнопки "Сфотографировать" должна включаться функция
	записи видео.При убратии пальца видео будет записано.После 
	записи видео,его можно будет предварительно просмотреть или удалить,а так же
	оставить подпись перед отправкой в ветку диалога 

	- В пользовательском интерфейсе "Камера" должно быть возможно сменить камеру с основной
	на фронтальную,и наоборот нажатием на кнопку "Сменить камеру".

	- В пользовательском интерфейсе "Камера" должна быть возможность выбрать фото из галереи,
	нажатием на фото из строки галереи.Для открытия всего списка фотографий,
	нужно провести палецем вверх по экрану.

--------------------------- Прикрепить ------------------------------

- Должна быть возможность делиться необходимыми файлами/данными с собеседником.
При нажатии кнопки "Прикрепить" пользователю должно открыться всплывающее
окно с кнопками для выбора файлов/данных которые он может отправить в ветку
диалога.Таких кнопок 6,а именно:

	1) "Документ".При нажатии на кнопку "Документ" должен открыться пользовательский
	Пользовательский интерфейс (файловый менеджер)со всеми имеющимися на
	девайсе документами для отправки.

	2) "Камера".При нажатии кнопки...смотреть "Камера"

	3) "Галерея".При нажатии кнопки "Галерея" пользователю
	должна открыться галерея со всеми имеющимися на девайсе фотографиями  
	пользователя для отправки.

	4) "Аудиофайлы".При нажатии кнопки "Аудиофайлы" пользователю
	должен открыться стандартный аудиоплеер со списком имеющихся на девайсе
	аудиозаписей для отправки

	5) "Мое местоположение".При нажатии на кнопку "Мое местоположение"
	пользователю,при наличии включенной геолокации и при наличии интернета,
	должна открыться карта с его местоположением для отправки

	6) "Контакты".При нажатии на кнопку "Контакты" пользователю должны открыться
	все имеющиеся в записной книжке контакты для отправки

----------------------------- Эмоджи ---------------------------------

- Для более быстрого и исчерывающего ответа собеседнику,в ветке диалога
присутствуют Эмоджи на любой выбор.Находятся слева от поля ввода текста.
При нажатии на Эмоджи,оно должно отобразиться в поле ввода текста,где нажатием
кнопки отправить,эмоджи появиться в ветке диалога.
